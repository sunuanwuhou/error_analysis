- 编辑态主工具收敛为：撤销 / 清空 / 退出

## 验收步骤
1. 打开任意题卡，确认右上角出现“画布”按钮
2. 点击进入后，只看到撤销 / 清空 / 更多 / 退出
3. 在题目上圈画后，入口状态应切换为“自动保存中”，随后变为“已保存”
4. 点击“更多 -> 导出图片”，应优先导出含题目与笔迹的图片
5. 退出后再次进入同题，笔迹应自动恢复
